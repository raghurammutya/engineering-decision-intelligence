# Production Break-Glass

## Trigger

An urgent production condition requires human intervention outside normal
autonomous enforcement or promotion flow.

## Standard Remediation

1. Declare break-glass mode with target, owner, reason, and expected duration.
2. Disable or pause affected autonomous production actions.
3. Make only the minimum required production change.
4. Record evidence for the action, reviewer, and rollback plan.
5. Restore normal controls after validation passes.

## Completion Evidence

- break-glass reason recorded,
- owner and reviewer recorded,
- autonomous action boundary restored,
- rollback or mitigation evidence linked,
- post-incident reconciliation completed.
