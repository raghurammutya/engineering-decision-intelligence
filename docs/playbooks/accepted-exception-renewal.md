# Accepted Exception Renewal

## Trigger

An artifact is allowed as an accepted exception rather than a canonical path.

## Standard Remediation

1. Confirm the exception still exists for a valid reason.
2. Confirm it remains non-mutating or tightly bounded.
3. Assign an owner.
4. Set or review renewal evidence.
5. Retire the exception if it is no longer needed.

## Completion Evidence

- exception reason,
- owner,
- renewal date or review evidence,
- replacement plan when applicable.
