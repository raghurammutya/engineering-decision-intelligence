# Database Migration Script

## Trigger

A script can alter database state or run migrations.

## Standard Remediation

1. Assign a database/platform owner.
2. Classify target environments.
3. Require rollback evidence.
4. Require dry-run or preflight mode where practical.
5. Keep production execution under controlled admission.

## Completion Evidence

- migration owner,
- target database/environment list,
- rollback procedure,
- validation report.
